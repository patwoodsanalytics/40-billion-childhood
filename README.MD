# A $40 Billion Childhood

*A public-interest analytics project examining the commercialization, consolidation, and rising costs of youth sports in the United States.*

## Overview

Youth sports in America have evolved into a massive economic ecosystem involving:
- travel tournaments,
- elite club teams,
- recruiting platforms,
- private coaching,
- social media exposure,
- and investor-backed consolidation.

This project explores how rising costs and commercialization are reshaping childhood athletics through:
- public datasets,
- data visualization,
- systems analysis,
- and long-form writing.

The project accompanies an essay-in-progress titled:

> **A $40 Billion Childhood**

which examines how youth sports shifted from locally organized recreation into a highly commercialized national industry.

---

## Project Goals

This repository aims to:
- visualize trends in youth sports participation and spending,
- analyze access and income disparities,
- document consolidation within youth sports infrastructure,
- and make underlying data sources publicly accessible.

The goal is not to criticize youth sports themselves.

At their best, youth sports provide:
- mentorship,
- discipline,
- belonging,
- teamwork,
- and community.

This project instead asks:
> What happens when access to those experiences increasingly depends on wealth, travel, and commercialization?
